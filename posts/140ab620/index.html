Open函数：

python中利用open函数来实现对文件的读取和更改

这里我们用open()函数打开一个文件，注意open函数的返回值为一个对象，之后，我们对这个对象使用read方法来实现对文档的读取
下面是对于传入的第二个参数的解释，其决定了open的操作模式
| 操作模式 | 具体含义 |
| ——– | ——————————– |
| 'r' | 读取 （默认） |
| 'w' | 写入（会先截断之前的内容） |
| 'x' | 写入，如果文件已经存在会产生异常 |
| 'a' | 追加，将内容写入到已有文件的末尾 |
| 'b' | 二进制模式 |
| 't' | 文本模式（默认） |
| '+' | 更新（既可以读又可以写） |
